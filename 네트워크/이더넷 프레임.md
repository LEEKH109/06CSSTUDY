# 1. 이더넷 프레임

> 데이터 링크 계층에서 이더넷 프레임을 통해 데이터의 에러를 검출하고 캡슐화

> 물리 계층 헤더

- Preamble : 시작
- SFD : 이후부터 MAC 주소가 온다는 것을 알리는 코드
  > MAC 프레임
- DMAC : 수신 MAC
- SMAC : 송신 MAC
- EtherType : IP프로토콜을 정의 ex) IPv6
- Payload : 전달받은 데이터
- CRC : 에러 확인 비트
